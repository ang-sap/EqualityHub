# EqualityHub

## Project Overview

EqualityHub is an SDG-based educational website developed for the Web Systems and Technologies course. The project focuses on promoting Gender Equality (Sustainable Development Goal 5) by providing educational resources, support services, and community engagement features for students.

## Purpose

The website aims to:

* Promote awareness of gender equality in schools.
* Educate users about gender discrimination and stereotypes.
* Provide support resources for students.
* Offer a safe space for discussion and sharing experiences.
* Allow users to report incidents related to discrimination.

## Features

* Home Page
* Learn & About Page
* Community & Support Page
* Safe Space Forum
* Incident Reporting System
* General Feedback Form
* Admin Dashboard
* Educational Stories and Videos
